# 1. var is old and has function scope, let is newer with block scope, and const is also block-scoped but cannot be reassigned. Think of var as global/function-wide, let as local to {}, and const as fixed once set. let and const released on ES6.

# 2. The spread operator (...) simply expands arrays or objects into individual elements or properties.

# 3. map() → transforms each item and returns a new array. filter() → keeps only items that match a condition, returns a new array. forEach() → just runs a function on each item, returns nothing.

# 4. An arrow function is a shorter way to write functions in JavaScript. It’s released on ES6. It’s concise, doesn’t bind its own this, and is mainly used for cleaner, modern code.

# 5. Template literals are a modern way to create strings in JavaScript using backticks ` instead of quotes. It’s also released on ES6.
